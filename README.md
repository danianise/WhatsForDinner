# WhatsForDinner

A collection of favorite dinner recipes. The recipes can be filtered by different criteria, such as ingredients or (lack of) allergens.

## Languages used:
- HTML
- EJS
- CSS
- JavaScript
- Express
- Node.js
- Mongoose
